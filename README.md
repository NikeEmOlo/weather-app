#Weather App Project
####Completed as part of [The Odin Project](https://theodinproject.org) - full stack development course

##The intention
This project is designed to help me demonstrate and practice my ability to:
1. Develop asynchronous code
2. Fetch data using APIs
3. Utilise fetched data

##APIs used:
1. [Geoapify API](https://www.geoapify.com/) 
    1. Used for search suggestions
    2. Used to obtain latitude,longitude values for each city

2. [Visual Crossing API](https://www.visualcrossing.com/)
    1. Used to obtain current weather data at the time of search.
